# -NALDA
한국항공대학교 NALDA 결과물 제출
--------------------------------------------
프로젝트 소개

오픈소스 게임 AssaultCube의 메모리 구조를 Cheat Engine으로 분석하고, 그 결과로 찾아낸 메모리 주소를 활용하여 게임 내 특정 값(체력, 탄약)을 자동으로 변경하는 C# 프로그램임. 치트엔진이 내부적으로 수행하는 메모리 읽기/쓰기 과정을 Windows API로 구현함.
--------------------------------------------
사전 준비

프로그램을 실행하기 전에 아래 항목이 준비되어야 함.
1. AssaultCube 및 Cheat Engine 설치.
2. Cheat Engine으로 메모리 분석 완료 - Cheat Engine으로 게임의 체력 및 탄약 값에 해당하는 포인터 주소(정적 주소)와 오프셋을 미리 찾아 두어야 함
--------------------------------------------
포인터 정보

항목포인터 체인체력[["ac_client.exe"+0x0018009C] + 0xEC]탄약[[[["ac_client.exe"+0x0018009C] + 0x36C] + 0x14] + 0x0]

정적 베이스: "ac_client.exe" + 0x0018009C
체력 오프셋: 0xEC
탄약 오프셋 체인: 0x36C → 0x14 → 0x0
--------------------------------------------
개발 및 실행 환경

언어: C# (.NET / Windows Forms)
IDE: Visual Studio
빌드 플랫폼: x86 (32비트) — AssaultCube가 32비트 프로그램이므로 x86으로 빌드해야 함.
---------------------------------------------
실행 방법

1. AssaultCube를 실행.
2. Visual Studio 관리자 권한으로 실행.
3. 변경하고 싶은 값에 해당하는 버튼(탄약, 체력) 클릭.
